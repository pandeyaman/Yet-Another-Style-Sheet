CSS Grid Practice

Grids Elements

A grid element consist of a parent element with one or multiple child elements. The display property should be set as grid to make the container as a grid container. 

            
                {
                    display: grid | inline-grid;
                }

All direct children of the grid container automatically become grid items.

Spanning Grid to multiple columns and rows

A grid container contains grid items.
Genrally, the grid container containes one item per grid, such as one item per row and column. But by using grid-column | grid-row property we can specify the number of rows and columns the grid should span over.

• The 'grid-column' property

The grid-column property defines on which columns to place an item.
It takes two values: from which line to start the item-grid and on which line to end. 

                   
                .item1
                       {
                          grid-column: 1/5;
                       }

This will start the grid item from 1st columne line and end it at 5th column line, spanning the item-grid to 4 columns

• The 'grid-row' property

The grid-row property defines on which rows to place an item.
It takes two values: from which line to start the item-grid and on which line to end. 

                   
                .item1
                       {
                          grid-row: 2/4;
                       }

This will start the grid item from 2nd row line and end it at 4th row line, spanning the item-grid to 2 rows.

• The 'grid-area' property

The grid-area property is used as a short-hand notation for combining grid-column and grid-row 

                   
                .item1
                {
                    grid-area: 1/2/3/4;
                }

This will start the item from row-line-1 and column-line-2 and end the item at row-line-3 and column-line-4.
